## List User Workspaces

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/workspaces/?limit=10" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "message": "Workspaces retrieved successfully",
  "data": {
    "workspaces": [
      {
        "id": "workspace123",
        "name": "My Water Quality Project",
        "owner": "user123",
        "type": "private",
        "rol": "owner"
      }
    ],
    "next_index": "eyJpZCI6Im5leHQifQ=="
  }
}
```
</ResponseExample>

Retrieve all workspaces owned by the authenticated user.

**Endpoint:** `GET /workspaces/`

**Authentication:** Required (Bearer token)

### Query Parameters

<ParamField query="limit" type="number" default="10">
  Number of items to retrieve (1-100)
</ParamField>

<ParamField query="index" type="string">
  Pagination index for next page
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  Workspace data with pagination
  <Expandable title="Data Object">
    <ResponseField name="workspaces" type="array">
      Array of workspace objects
    </ResponseField>
    <ResponseField name="next_index" type="string">
      Index for next page (if available)
    </ResponseField>
  </Expandable>
</ResponseField>

## Get All Workspaces (Admin)

<RequestExample>
```bash cURL
curl -X GET https://api.example.com/workspaces/all/ \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "message": "All workspaces retrieved successfully",
  "workspaces": [
    {
      "id": "workspace123",
      "name": "Production Workspace",
      "owner": "user123",
      "type": "private",
      "created_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```
</ResponseExample>

Retrieve all workspaces in the system. This endpoint is restricted to admin users only.

**Endpoint:** `GET /workspaces/all/`

**Authentication:** Required (Admin role only)

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="workspaces" type="array">
  Array of all workspace objects in the system
  <Expandable title="workspace object">
    <ResponseField name="id" type="string">
      Workspace unique identifier
    </ResponseField>
    <ResponseField name="name" type="string">
      Workspace name
    </ResponseField>
    <ResponseField name="owner" type="string">
      Owner user ID
    </ResponseField>
    <ResponseField name="type" type="string">
      Workspace type (private or public)
    </ResponseField>
  </Expandable>
</ResponseField>

<Warning>
  This endpoint requires admin privileges. Regular users will receive a 403 Forbidden error.
</Warning>

## Create Workspace

```bash cURL
  -H "Content-Type: application/json" \

    "type": "private"
  }'
```
</RequestExample>

<ResponseExample>
```json Success

  "data": {

    "owner": "user123",
```

Create a new workspace.

**Endpoint:** `POST /workspaces/`

## Add Guest to Workspace

<RequestExample>
```bash cURL
curl -X POST https://api.example.com/workspaces/workspace123/guest/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "guest": "guest@example.com",
    "rol": "viewer"
  }'
```
</RequestExample>

<ResponseExample>
```json Success
{
  "message": "Guest created successfully",
  "guest": {
    "uid": "guest123",
    "email": "guest@example.com",
    "username": "guestuser",
    "rol": "viewer"
  }
}
```
</ResponseExample>

Add a guest (collaborator) to a workspace. An invitation email will be sent.

**Endpoint:** `POST /workspaces/{id}/guest/`

**Authentication:** Required (Bearer token, must be owner)

### Path Parameters

<ParamField path="id" type="string" required>
  Workspace ID
</ParamField>

### Request Body

<ParamField body="guest" type="string" required>
  Email address of the user to invite
</ParamField>

<ParamField body="rol" type="string" required>
  Role for the guest: `viewer`, `editor`, or `admin`
</ParamField>
